## /cat Endpoint

GET
DELETE - deletes first cat in the queue and returns the next cat

## /dog Endpoint

GET
DELETE - deletes first dog in the queue and returns the next dog

## /people Endpoint

GET
DELETE
body: person: xxx
returns all people in queue

POST
body: person: xxx
